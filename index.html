Premium SMS and stands for Premium Short Message Service. PSMS works exactly the same as Premium Rate SMS and Premium SMS and offers the same billing capabilities and functionality. KAMCENT TECHNOLOGIES LTD offers its customers Shortcodes in numerous countries and for many different purposes. KAMCENT TECHNOLOGIES LTD customers use shortcodes for the billing of mobile content, value added services, mobile marketing and sms campaigns. Shortcodes are 3-8 digit telephone numbers provisioned on a network operators network that can be used to receive or send sms or premium sms messages. The Two Way Messaging application is a unique application developed with the aim to service a wide range of short code based applications. In it you can connect to various providers using the vendor independent SMPP (Short Message Peer to Peer) or HTTP (Hyper Text Transfer Protocol) protocol for sending and receiving messages.
